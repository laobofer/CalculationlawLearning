## 1. 数组常见处理方法总结

| 技巧 | 核心思想 | 作用 | 典型应用 |
|------|----------|------|----------|
| **双指针（同向）** | 快慢指针，一个遍历一个填充 | 原地去重、删除指定元素 | 删除排序数组重复项、移动零 |
| **双指针（相向）** | 左右指针向中间移动 | N 数求和 | 两数之和、三数之和 |
| **前缀和** | `preSum[i] = sum(nums[0..i))` | O(1) 时间求区间和/积 | 区域和检索、和为 K 的子数组 |
| **前缀和 + 哈希表** | 存前缀和到索引/次数的映射 | 求满足条件的子数组个数 | 和为 K 的子数组、连续数组 |
| **差分数组** | 记录相邻元素差值 | 快速对区间做加减法 | 航班预订统计、拼车 |
| **滑动窗口** | 维护 left/right 边界，动态伸缩窗口 | 求最长/最短连续子数组/子串 | 无重复字符最长子串、最小覆盖子串 |
| **二分搜索** | 每次排除一半搜索空间 | 有序数组快速查找 | 搜索插入位置、搜索旋转排序数组 |
| **二分答案（值域二分）** | 对值做二分，判定可行性 | 求最小化最大值 | 珂珂吃香蕉、分割数组 |
| **桶思想** | 将元素按大小分配到桶中 | 快速判断值是否在范围内 | 存在重复元素 III |

## 2. 双指针

### 2.1 两数之和

- **题目**：[1. 两数之和](https://leetcode.cn/problems/two-sum)
- **描述**：在数组中找出两个数，和为 target，返回它们的下标。
- **示例**：
  - 输入：`nums = [2,7,11,15]`, `target = 9`
  - 输出：`[0,1]`

#### 原始实现

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        i, j = 0, len(nums) - 1
        temp = sorted(enumerate(nums), key=lambda x: x[1])
        while i < j:
            sum = temp[i][1] + temp[j][1]
            if sum > target:
                j -= 1
            elif sum < target:
                i += 1
            else:
                return [temp[i][0], temp[j][0]]
        return []
```

#### 标准实现（哈希表法 — ⭐推荐）

> **评价**：你的排序 + 相向双指针法可行，但破坏了原始下标。更通用的方法是 **哈希表**，一次遍历即可。

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        seen = {}
        for i, num in enumerate(nums):
            complement = target - num
            if complement in seen:
                return [seen[complement], i]
            seen[num] = i
        return []
```

> **考试建议**：哈希表法 O(n) 一次遍历，无需排序，**强烈推荐**。排序双指针法适用于"返回数值而非下标"的场景（如三数之和）。

### 2.2 三数之和

- **题目**：[15. 三数之和](https://leetcode.cn/problems/3sum)
- **描述**：找出数组中所有和为 0 且不重复的三元组。
- **示例**：
  - 输入：`nums = [-1,0,1,2,-1,-4]`
  - 输出：`[[-1,-1,2],[-1,0,1]]`

#### 原始实现

```python
class Solution:
    def threeSum(self, nums: list[int]) -> list[list[int]]:
        return self.threeSumTarget(nums, 0)

    def threeSumTarget(self, nums: List[int], target: int) -> List[List[int]]:
        nums.sort()
        res = []
        i = 0
        while i < len(nums):
            pairs = self.twoSumTarget(nums, i + 1, target - nums[i])
            for pair in pairs:
                pair.append(nums[i])
                res.append(pair)
            # 跳过重复元素
            for j in range(i + 1, len(nums)):
                if nums[j] == nums[i]:
                    i += 1
                else:
                    break
            i += 1
        return res

    def twoSumTarget(self, nums, start, target):
        lo, hi = start, len(nums) - 1
        res = []
        while lo < hi:
            sum = nums[lo] + nums[hi]
            if sum > target:
                hi -= 1
            elif sum < target:
                lo += 1
            else:
                res.append([nums[lo], nums[hi]])
                # 跳过重复
                for i in range(lo + 1, hi):
                    if nums[i] == nums[lo]:
                        lo += 1
                    else:
                        break
                for i in range(hi - 1, lo, -1):
                    if nums[i] == nums[hi]:
                        hi -= 1
                    else:
                        break
                lo += 1
                hi -= 1
        return res
```

#### 标准实现（⭐推荐）

> **评价**：你的"泛化 nSumTarget"思路很好（可扩展为四数之和）。下面给出更紧凑的版本：

```python
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        nums.sort()
        res = []
        n = len(nums)
        for i in range(n):
            if i > 0 and nums[i] == nums[i - 1]:
                continue
            # 两数之和（相向双指针）
            lo, hi = i + 1, n - 1
            while lo < hi:
                total = nums[i] + nums[lo] + nums[hi]
                if total < 0:
                    lo += 1
                elif total > 0:
                    hi -= 1
                else:
                    res.append([nums[i], nums[lo], nums[hi]])
                    lo += 1
                    hi -= 1
                    while lo < hi and nums[lo] == nums[lo - 1]:
                        lo += 1
                    while lo < hi and nums[hi] == nums[hi + 1]:
                        hi -= 1
        return res
```

### 2.3 四数之和

- **题目**：[18. 四数之和](https://leetcode.cn/problems/4sum)
- **描述**：找出数组中所有和为 target 且不重复的四元组。
- **示例**：
  - 输入：`nums = [1,0,-1,0,-2,2]`, `target = 0`
  - 输出：`[[-2,-1,1,2],[-2,0,0,2],[-1,0,0,1]]`

#### 原始实现

```python
# 思路：在 threeSumTarget 外再套一层循环，与三数之和结构相同
class Solution:
    def fourSum(self, nums: List[int], target: int) -> List[List[int]]:
        nums.sort()
        res = []
        i = 0
        while i < len(nums):
            pairs = self.threeSumTarget(nums, i + 1, target - nums[i])
            for pair in pairs:
                pair.append(nums[i])
                res.append(pair)
            for j in range(i + 1, len(nums)):
                if nums[j] == nums[i]:
                    i += 1
                else:
                    break
            i += 1
        return res
    # ... （threeSumTarget 和 twoSumTarget 同上）
```

#### 标准实现

> **评价**：利用 nSum 递归泛化是经典思路。考试中如果只考四数之和，可以直接在三数之和外套一层循环：

```python
class Solution:
    def fourSum(self, nums: List[int], target: int) -> List[List[int]]:
        nums.sort()
        res = []
        n = len(nums)
        for i in range(n):
            if i > 0 and nums[i] == nums[i - 1]:
                continue
            for j in range(i + 1, n):
                if j > i + 1 and nums[j] == nums[j - 1]:
                    continue
                lo, hi = j + 1, n - 1
                while lo < hi:
                    total = nums[i] + nums[j] + nums[lo] + nums[hi]
                    if total < target:
                        lo += 1
                    elif total > target:
                        hi -= 1
                    else:
                        res.append([nums[i], nums[j], nums[lo], nums[hi]])
                        lo += 1
                        hi -= 1
                        while lo < hi and nums[lo] == nums[lo - 1]:
                            lo += 1
                        while lo < hi and nums[hi] == nums[hi + 1]:
                            hi -= 1
        return res
```

### 2.4 删除有序数组中的重复项

- **题目**：[26. 删除有序数组中的重复项](https://leetcode.cn/problems/remove-duplicates-from-sorted-array)
- **描述**：原地删除有序数组中的重复项，使每个元素只出现一次，返回新长度。
- **示例**：
  - 输入：`nums = [0,0,1,1,1,2,2,3,3,4]`
  - 输出：`5`, `nums = [0,1,2,3,4,...]`

#### 原始实现

```python
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        slow, fast = 0, 0
        target = -101
        ret = 0
        while fast < len(nums):
            if target != nums[fast]:
                nums[slow] = nums[fast]
                target = nums[fast]
                slow += 1
                ret += 1
            fast += 1
        return ret
```

#### 标准实现（⭐推荐）

> **评价**：快慢指针，思路清晰。更简洁的写法：

```python
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        if not nums:
            return 0
        slow = 0
        for fast in range(1, len(nums)):
            if nums[fast] != nums[slow]:
                slow += 1
                nums[slow] = nums[fast]
        return slow + 1
```

> **考试推荐**：快慢指针是处理"原地修改"类问题的标配，**强烈推荐**。

### 2.5 删除有序数组中的重复项 II

- **题目**：[80. 删除有序数组中的重复项 II](https://leetcode.cn/problems/remove-duplicates-from-sorted-array-ii)
- **描述**：原地删除，使每个元素最多出现两次，返回新长度。
- **示例**：
  - 输入：`nums = [1,1,1,2,2,3]`
  - 输出：`5`, `nums = [1,1,2,2,3]`

#### 原始实现

```python
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        slow, fast = 0, 0
        cur_num, num, ret = 10001, 1, 0
        while fast < len(nums):
            if cur_num != nums[fast]:
                cur_num = nums[fast]
                nums[slow] = cur_num
                slow += 1
                num = 1
                ret += 1
            else:
                if num == 1:
                    nums[slow] = cur_num
                    slow += 1
                    num = 1
                    ret += 1
                num += 1
            fast += 1
        return ret
```

#### 标准实现（⭐推荐）

> **评价**：你的实现功能正确，但逻辑稍显复杂。一个更通用的技巧：因为只允许最多出现两次，直接比较 `nums[slow-2]` 与当前元素：

```python
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        slow = 0
        for num in nums:
            if slow < 2 or num != nums[slow - 2]:
                nums[slow] = num
                slow += 1
        return slow
```

> 扩展：允许重复 k 次，只需将 `2` 改为 `k`。**面试中非常实用的技巧**。

### 2.6 移动零

- **题目**：[283. 移动零](https://leetcode.cn/problems/move-zeroes)
- **描述**：将所有 0 移动到数组末尾，保持非零元素相对顺序。
- **示例**：
  - 输入：`nums = [0,1,0,3,12]`
  - 输出：`[1,3,12,0,0]`

#### 原始实现

```python
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        slow, fast = 0, 0
        while fast < len(nums):
            if nums[fast] != 0:
                nums[slow], nums[fast] = nums[fast], nums[slow]
                slow += 1
            fast += 1
```

#### 标准实现（⭐推荐）

> **评价**：你的实现就是最优解！快慢指针，快指针遇非零就与慢指针交换。时间 O(n)，空间 O(1)。

```python
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        slow = 0
        for fast in range(len(nums)):
            if nums[fast] != 0:
                nums[slow], nums[fast] = nums[fast], nums[slow]
                slow += 1
```

### 2.8 有序数组的平方

- **题目**：[977. 有序数组的平方](https://leetcode.cn/problems/squares-of-a-sorted-array)
- **描述**：对有序数组（含负数）的每个数求平方，结果也按非递减顺序排列。
- **示例**：
  - 输入：`nums = [-4,-1,0,3,10]`
  - 输出：`[0,1,9,16,100]`

#### 原始实现

```python
class Solution:
    def sortedSquares(self, nums: List[int]) -> List[int]:
        i, j = 0, len(nums) - 1
        p = len(nums) - 1
        ret = [0 for _ in range(len(nums))]
        while i <= j:
            if nums[i] ** 2 >= nums[j] ** 2:
                ret[p] = nums[i] ** 2
                i += 1
            else:
                ret[p] = nums[j] ** 2
                j -= 1
            p -= 1
        return ret
```

#### 标准实现（⭐推荐）

> **评价**：相向双指针，**这就是标准解法**，最优美。时间 O(n)，空间 O(1)（不计返回数组）。

```python
class Solution:
    def sortedSquares(self, nums: List[int]) -> List[int]:
        n = len(nums)
        res = [0] * n
        i, j, k = 0, n - 1, n - 1
        while i <= j:
            if abs(nums[i]) > abs(nums[j]):
                res[k] = nums[i] * nums[i]
                i += 1
            else:
                res[k] = nums[j] * nums[j]
                j -= 1
            k -= 1
        return res
```

### 2.9 合并两个有序数组

- **题目**：[88. 合并两个有序数组](https://leetcode.cn/problems/merge-sorted-array)
- **描述**：将 nums2 合并进 nums1（nums1 空间足够大），使结果有序。
- **示例**：
  - 输入：`nums1 = [1,2,3,0,0,0]`, `m = 3`, `nums2 = [2,5,6]`, `n = 3`
  - 输出：`nums1 = [1,2,2,3,5,6]`

#### 原始实现

```python
class Solution:
    def merge(self, nums1: List[int], m: int, nums2: List[int], n: int) -> None:
        index_1, index_2 = m - 1, n - 1
        index = m + n - 1
        while index_1 >= 0 and index_2 >= 0:
            max_ = -1e10
            if nums1[index_1] >= nums2[index_2]:
                max_ = nums1[index_1]
                index_1 -= 1
            else:
                max_ = nums2[index_2]
                index_2 -= 1
            nums1[index] = max_
            index -= 1
        while index_1 >= 0:
            nums1[index] = nums1[index_1]
            index_1 -= 1
            index -= 1
        while index_2 >= 0:
            nums1[index] = nums2[index_2]
            index_2 -= 1
            index -= 1
```

#### 标准实现（⭐推荐）

> **评价**：**从后往前填充**是最优解法，避免了从前填充需要移动元素的问题。简洁版本：

```python
class Solution:
    def merge(self, nums1: List[int], m: int, nums2: List[int], n: int) -> None:
        p1, p2, p = m - 1, n - 1, m + n - 1
        while p1 >= 0 and p2 >= 0:
            if nums1[p1] > nums2[p2]:
                nums1[p] = nums1[p1]
                p1 -= 1
            else:
                nums1[p] = nums2[p2]
                p2 -= 1
            p -= 1
        # 只需处理 nums2 剩余（因为 nums1 剩余已经在正确位置）
        nums1[:p2 + 1] = nums2[:p2 + 1]
```

### 2.11 颜色分类

- **题目**：[75. 颜色分类](https://leetcode.cn/problems/sort-colors)
- **描述**：对只含 0、1、2 的数组原地排序（荷兰国旗问题）。
- **示例**：
  - 输入：`nums = [2,0,2,1,1,0]`
  - 输出：`[0,0,1,1,2,2]`

#### 原始实现

```python
class Solution:
    def sortColors(self, nums: List[int]) -> None:
        len_ = max(nums) + 1
        count = [0 for _ in range(len_)]
        for num in nums:
            count[num] += 1
        i = -1
        for index in count:
            for _ in range(index):
                i += 1
                nums[i] = index
```

#### 标准实现（三指针 — ⭐推荐）

> **评价**：计数排序可以，但需要两趟遍历。荷兰国旗问题的经典解法是用 **三指针** 一趟搞定。

```python
class Solution:
    def sortColors(self, nums: List[int]) -> None:
        zero, two = 0, len(nums) - 1
        i = 0
        while i <= two:
            if nums[i] == 0:
                nums[i], nums[zero] = nums[zero], nums[i]
                zero += 1
                i += 1
            elif nums[i] == 2:
                nums[i], nums[two] = nums[two], nums[i]
                two -= 1
            else:
                i += 1
```

> **考试建议**：三指针 O(n) 一趟遍历，面试中效果很好，**推荐理解**。如果时间紧张，计数排序也是一种选择。

### 2.12 寻找重复数

- **题目**：[287. 寻找重复数](https://leetcode.cn/problems/find-the-duplicate-number)
- **描述**：n+1 个数字在 [1, n] 范围，只有一个重复数，找出来（不能修改数组，O(1) 空间）。
- **示例**：
  - 输入：`nums = [1,3,4,2,2]`
  - 输出：`2`

#### 原始实现

```python
# 将数组视为链表：index -> nums[index]
class Solution:
    def next_node(self, nums, next):
        return nums[next]

    def next_next_node(self, nums, next):
        return nums[nums[next]]

    def findDuplicate(self, nums: List[int]) -> int:
        fast, slow = 0, 0
        while True:
            fast = self.next_next_node(nums, fast)
            slow = self.next_node(nums, slow)
            if slow == fast:
                break
        slow = 0
        while fast != slow:
            slow = self.next_node(nums, slow)
            fast = self.next_node(nums, fast)
        return slow
```

#### 标准实现（⭐推荐）

> **评价**：你的"视为链表找环入口"思路非常巧妙！这是最优解，时间 O(n)，空间 O(1)。

```python
class Solution:
    def findDuplicate(self, nums: List[int]) -> int:
        # Floyd 判圈
        slow = fast = nums[0]
        while True:
            slow = nums[slow]
            fast = nums[nums[fast]]
            if slow == fast:
                break
        slow = nums[0]
        while slow != fast:
            slow = nums[slow]
            fast = nums[fast]
        return slow
```

> 这道题还有二分法解法：在 [1, n] 上二分，统计 `<= mid` 的数的个数，但不如 Floyd 法高效。

## 3. 前缀和

> **核心思想**：`preSum[i] = nums[0] + nums[1] + ... + nums[i-1]`，区间 `[l, r]` 的和 = `preSum[r+1] - preSum[l]`。
>
> 可将 O(n) 的区间查询优化为 O(1)。

### 3.1 区域和检索 - 数组不可变

- **题目**：[303. 区域和检索 - 数组不可变](https://leetcode.cn/problems/range-sum-query-immutable)
- **描述**：多次查询 nums 的区间和。
- **示例**：
  - 输入：`nums = [-2,0,3,-5,2,-1]`, 查询 `[0,2]`, `[2,5]`, `[0,5]`
  - 输出：`1`, `-1`, `-3`

#### 原始实现

```python
class NumArray:
    def __init__(self, nums: List[int]):
        self.preSum = [0 for _ in range(len(nums) + 1)]
        for i in range(1, len(nums) + 1):
            self.preSum[i] = self.preSum[i - 1] + nums[i]

    def sumRange(self, left: int, right: int) -> int:
        return self.preSum[right + 1] - self.preSum[left]
```

#### ✅ 标准实现

> **评价**：**完全正确**，这就是前缀和的标准模板。

```python
class NumArray:
    def __init__(self, nums: List[int]):
        self.pre = [0]
        for num in nums:
            self.pre.append(self.pre[-1] + num)

    def sumRange(self, left: int, right: int) -> int:
        return self.pre[right + 1] - self.pre[left]
```

### 3.2 二维区域和检索 - 矩阵不可变

- **题目**：[304. 二维区域和检索 - 矩阵不可变](https://leetcode.cn/problems/range-sum-query-2d-immutable)
- **描述**：多次查询矩阵的子矩阵和。
- **示例**：
  - 输入：`matrix = [[3,0,1,4,2],[5,6,3,2,1],[1,2,0,1,5],[4,1,0,1,7],[1,0,3,0,5]]`
  - 查询：`[2,1,4,3]` → `8`

#### 原始实现

```python
class NumMatrix:
    def __init__(self, matrix: List[List[int]]):
        m, n = len(matrix) + 1, len(matrix[0]) + 1
        self.preSum = [[0 for _ in range(n)] for _ in range(m)]
        for i in range(1, m):
            for j in range(1, n):
                self.preSum[i][j] = self.preSum[i-1][j] + self.preSum[i][j-1] - self.preSum[i-1][j-1] + matrix[i-1][j-1]

    def sumRegion(self, row1: int, col1: int, row2: int, col2: int) -> int:
        return (
            self.preSum[row2+1][col2+1]
            - self.preSum[row1][col2+1]
            - self.preSum[row2+1][col1]
            + self.preSum[row1][col1]
        )
```

#### ✅ 标准实现

> **评价**：**完全正确**，二维前缀和的标准实现。容斥原理：`sum(大) - 左 - 上 + 左上`。

### 3.3 和为 K 的子数组

- **题目**：[560. 和为 K 的子数组](https://leetcode.cn/problems/subarray-sum-equals-k)
- **描述**：统计和为 K 的连续子数组的个数。
- **示例**：
  - 输入：`nums = [1,1,1]`, `k = 2`
  - 输出：`2`

#### 原始实现

```python
class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        n = len(nums) + 1
        preSum = [0] * n
        for i in range(1, n):
            preSum[i] = preSum[i - 1] + nums[i - 1]

        pos = {}
        for idx in range(n):
            val = preSum[idx]
            if val not in pos:
                pos[val] = []
            pos[val].append(idx)

        res = 0
        for i in range(1, n):
            need = preSum[i] - k
            if need in pos:
                cnt = bisect.bisect_left(pos[need], i)
                res += cnt
        return res
```

#### 标准实现（前缀和 + 哈希表 — ⭐推荐）

> **评价**：你用了"先建全表再二分"，时间复杂度 O(n log n)。更优的是 **边遍历边计数** 的在线方法 O(n)：

```python
class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        from collections import defaultdict
        pre_sum = 0
        count = defaultdict(int)
        count[0] = 1  # 初始：前缀和为 0 出现 1 次
        res = 0
        for num in nums:
            pre_sum += num
            # pre_sum - k 表示需要的前缀和
            res += count[pre_sum - k]
            count[pre_sum] += 1
        return res
```

> **核心**：`preSum[j] - preSum[i] = k` 等价于找前面有多少个 `preSum[j] - k`。边遍历边更新，**强烈推荐**。

### 3.4 和可被 K 整除的子数组

- **题目**：[974. 和可被 K 整除的子数组](https://leetcode.cn/problems/subarray-sums-divisible-by-k)
- **描述**：统计和能被 K 整除的连续子数组个数。
- **示例**：
  - 输入：`nums = [4,5,0,-2,-3,1]`, `k = 5`
  - 输出：`7`

#### 原始实现

```python
class Solution:
    def subarraysDivByK(self, nums: List[int], k: int) -> int:
        n = len(nums) + 1
        preSum = [0 for _ in range(n)]
        for i in range(1, n):
            preSum[i] = preSum[i - 1] + nums[i - 1]

        dict = defaultdict(list)
        for i in range(n):
            dict[preSum[i] % k].append(i)

        res = 0

            if need in dict:
                res += sum(1 for index in dict[need] if index < i)
        return res
```

#### 标准实现（⭐推荐）

> **评价**：与"和为 K 的子数组"类似，更优的是边遍历边计数：

```python
class Solution:
    def subarraysDivByK(self, nums: List[int], k: int) -> int:
        from collections import defaultdict
        pre_sum = 0
        mod_count = defaultdict(int)
        mod_count[0] = 1
        res = 0
        for num in nums:
            pre_sum += num
            mod = pre_sum % k
            res += mod_count[mod]
            mod_count[mod] += 1
        return res
```

> **关键**：`(preSum[j] - preSum[i]) % k == 0` 等价于 `preSum[j] % k == preSum[i] % k`。

### 3.5 寻找数组的中心下标

- **题目**：[724. 寻找数组的中心下标](https://leetcode.cn/problems/find-pivot-index)
- **描述**：找到下标 i，使得左侧元素和等于右侧元素和。
- **示例**：
  - 输入：`nums = [1,7,3,6,5,6]`
  - 输出：`3`

#### 原始实现

```python
class Solution:
    def pivotIndex(self, nums: List[int]) -> int:
        n = len(nums) + 1
        preSum = [0 for _ in range(n)]
        for i in range(1, n):
            preSum[i] = preSum[i - 1] + nums[i - 1]
        for i in range(1, n - 1):
            leftSum = preSum[i - 1] - preSum[0]
            rightSum = preSum[n - 1] - preSum[i]
            if leftSum == rightSum:
                return i - 1
        return -1
```

#### 标准实现（⭐推荐）

> **评价**：用前缀和即可，无需额外数组也可用"总和 + 左和"判断：

```python
class Solution:
    def pivotIndex(self, nums: List[int]) -> int:
        total = sum(nums)
        left_sum = 0
        for i, num in enumerate(nums):
            if left_sum == total - left_sum - num:
                return i
            left_sum += num
        return -1
```

### 3.7 除自身以外数组的乘积

- **题目**：[238. 除自身以外数组的乘积](https://leetcode.cn/problems/product-of-array-except-self)
- **描述**：返回数组 answer，其中 answer[i] 等于 nums 中除 nums[i] 之外其余各元素的乘积（不能用除法）。
- **示例**：
  - 输入：`nums = [1,2,3,4]`
  - 输出：`[24,12,8,6]`

#### 原始实现

```python
class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        n = len(nums)
        prefix = [nums[0] for _ in range(n)]
        suffix = [nums[n - 1] for _ in range(n)]
        answer = [0 for _ in range(n)]
        for i in range(1, n):
            prefix[i] = prefix[i - 1] * nums[i]
            suffix[i] = suffix[i - 1] * nums[n - 1 - i]
        suffix.reverse()
        for i in range(n):
            if i - 1 >= 0 and i + 1 <= n - 1:
                answer[i] = prefix[i - 1] * suffix[i + 1]
            elif i - 1 < 0:
                answer[i] = suffix[i + 1]
            elif i + 1 > n - 1:
                answer[i] = prefix[i - 1]
        return answer
```

#### 标准实现（⭐推荐）

> **评价**：你的思路正确——前缀积 × 后缀积。但可以优化为 **两次遍历 O(1) 额外空间**：

```python
class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        n = len(nums)
        answer = [1] * n

        # 从左到右：前缀积
        left = 1
        for i in range(n):
            answer[i] = left
            left *= nums[i]

        # 从右到左：后缀积
        right = 1
        for i in range(n - 1, -1, -1):
            answer[i] *= right
            right *= nums[i]

        return answer
```

> **考试推荐**：一次前缀 + 一次后缀，巧妙利用 answer 数组存储中间结果，空间 O(1)（不计输出）。

### 3.10 矩阵区域和

- **题目**：[1314. 矩阵区域和](https://leetcode.cn/problems/matrix-block-sum)
- **描述**：对矩阵每个位置 `(i,j)`，返回以 `(i,j)` 为中心、`k x k` 为半径的矩形区域内所有元素之和。
- **示例**：
  - 输入：`mat = [[1,2,3],[4,5,6],[7,8,9]]`, `k = 1`
  - 输出：`[[12,21,16],[27,45,33],[24,39,28]]`

#### 原始实现

```python
class Solution:
    def get_preSum(self, matrix):
        m, n = len(matrix) + 1, len(matrix[0]) + 1
        preSum = [[0 for _ in range(n)] for _ in range(m)]
        for i in range(1, m):
            for j in range(1, n):
                preSum[i][j] = preSum[i-1][j] + preSum[i][j-1] - preSum[i-1][j-1] + matrix[i-1][j-1]
        return preSum

    def sumRegion(self, preSum, row1, col1, row2, col2):
        return preSum[row2+1][col2+1] - preSum[row1][col2+1] - preSum[row2+1][col1] + preSum[row1][col1]

    def matrixBlockSum(self, mat: List[List[int]], k: int) -> List[List[int]]:
        preSum = self.get_preSum(mat)
        m, n = len(mat), len(mat[0])
        res = [[0 for _ in range(n)] for _ in range(m)]
        for i in range(m):
            for j in range(n):
                res[i][j] = self.getSum(preSum, i-k, j-k, i+k, j+k, m-1, n-1)
        return res
```

#### ✅ 标准实现

> **评价**：**完全正确**，二维前缀和的经典应用。注意边界处理即可。

### 3.11 表现良好的时间段

- **题目**：[1124. 表现良好的时间段](https://leetcode.cn/problems/longest-well-performing-interval)
- **描述**：若工作时长 > 8 小时记 +1，否则记 -1，要求和 > 0 的最长区间长度。
- **示例**：
  - 输入：`hours = [9,9,6,0,6,6,9]`
  - 输出：`3`

#### 原始实现

```python
class Solution:
    def longestWPI(self, hours: List[int]) -> int:
        n = len(hours) + 1
        preSum = [0 for _ in range(n)]
        for i in range(1, n):
            preSum[i] = preSum[i-1] + (1 if hours[i-1] > 8 else -1)
        dict = defaultdict(list)
        for i in range(n):
            dict[preSum[i]].append(i)
        res = 0
        for i in range(1, n):
            need = preSum[i] - 1
            if preSum[i] > 0:
                res = max(res, i)
            if need in dict:
                res = max(res, i - dict[need][0])
```

#### 标准实现（优化版）

## 4. 差分数组

> **核心思想**：`diff[i] = nums[i] - nums[i-1]`，对区间 `[l, r]` 加 val 只需 `diff[l] += val`、`diff[r+1] -= val`。
>
> 可将 O(n) 的区间加法优化为 O(1)。

### 4.1 区间加法

- **题目**：[370. 区间加法](https://leetcode.cn/problems/range-addition)（LeetCode VIP）
- **描述**：给定一个长度为 length 的零数组，执行若干次区间加法操作。
- **示例**：
  - 输入：`length = 5`, `updates = [[1,3,2],[2,4,3],[0,2,-2]]`
  - 输出：`[-2,0,3,5,3]`

#### 原始实现

```python
class Solution:
    def increase(self, diff, start_index, end_index, inc):
        if start_index >= 0 and start_index < len(diff):
            diff[start_index] += inc
        if end_index + 1 < len(diff):
            diff[end_index + 1] -= inc

    def get_nums(self, diff):
        res = []
        res.append(diff[0])
        for i in range(1, len(diff)):
            res.append(diff[i] + res[i - 1])
        return res

    def getModifiedArray(self, length: int, updates: List[List[int]]) -> List[int]:
        diff = [0 for _ in range(length)]
        for update in updates:
            start_index, end_index, inc = update
            self.increase(diff, start_index, end_index, inc)
        return self.get_nums(diff)
```

#### ✅ 标准实现

> **评价**：**完全正确**，差分数组的标准模板。

```python
class Solution:
    def getModifiedArray(self, length: int, updates: List[List[int]]) -> List[int]:
        diff = [0] * length
        for l, r, val in updates:
            diff[l] += val
            if r + 1 < length:
                diff[r + 1] -= val

        res = [0] * length
        cur = 0
        for i in range(length):
            cur += diff[i]
            res[i] = cur
        return res
```

## 5. 滑动窗口

> **核心框架**：
> ```
> left, right = 0, 0
> while right < len(s):
>     扩大窗口（移入 right 元素）
>     right += 1
>     while 需要收缩窗口:
>         缩小窗口（移出 left 元素）
>         left += 1
>     更新答案
> ```
>
> 滑动窗口适用于求 **最长/最短/连续** 子数组/子串的问题。

### 5.1 无重复字符的最长子串

- **题目**：[3. 无重复字符的最长子串](https://leetcode.cn/problems/longest-substring-without-repeating-characters)

- **示例**：
  - 输入：`s = "abcabcbb"`
  - 输出：`3`（"abc"）

#### 原始实现

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        from collections import defaultdict
        window = defaultdict(int)
        left, right = 0, 0
        res = 0
        while right < len(s):
            chr = s[right]
            window[chr] += 1
            right += 1
            if window[chr] == 1:
                res = max(right - left, res)
            while left < right and window[chr] >= 2:
                chr_ = s[left]
                window[chr_] -= 1
                left += 1
        return res
```

#### 标准实现（⭐推荐）

> **评价**：收缩条件只需看当前字符（新加入导致重复）：


### 5.4 最小覆盖子串

- **题目**：[76. 最小覆盖子串](https://leetcode.cn/problems/minimum-window-substring)
- **描述**：在 s 中找到覆盖 t 中所有字符的最短子串。
- **示例**：
  - 输入：`s = "ADOBECODEBANC"`, `t = "ABC"`
  - 输出：`"BANC"`

#### 原始实现

```python
class Solution:
    def minWindow(self, s: str, t: str) -> str:
        from collections import defaultdict
        window = defaultdict(int)
        need = defaultdict(int)
        for chr in t:
            need[chr] += 1
        left, right = 0, 0
        res = [0, 0]
        while right < len(s):
            chr = s[right]
            window[chr] += 1
            right += 1
            while left < right and self.is_right_substr(window, need):
                res[:] = [left, right] if (right - left) < (res[1] - res[0]) or res == [0, 0] else res
                chr = s[left]
                left += 1
                window[chr] -= 1
        return s[res[0]:res[1]]
```

#### 标准实现（⭐推荐）

> **评价**：**match 变量**是性能关键——避免每次全表扫描。

```python
class Solution:
    def minWindow(self, s: str, t: str) -> str:
        from collections import Counter
        need = Counter(t)
        window = {}
        left = match = 0
        start, min_len = 0, float('inf')

        for right, ch in enumerate(s):
            window[ch] = window.get(ch, 0) + 1
            if ch in need and window[ch] == need[ch]:
                match += 1

            while match == len(need):
                if right - left + 1 < min_len:
                    start = left
                    min_len = right - left + 1
                window[s[left]] -= 1
                if s[left] in need and window[s[left]] < need[s[left]]:
                    match -= 1
                left += 1

        return "" if min_len == float('inf') else s[start:start + min_len]
```

### 5.11 将 x 减到 0 的最小操作数

- **描述**：每次从数组最左或最右移除一个数，求最小操作次数使移除数的和 = x。
- **示例**：

  - 输出：`2`

#### 原始实现

```python

    def minOperations(self, nums: List[int], x: int) -> int:
        window = 0
        need = sum(nums) - x
        if need == 0:
            return len(nums)
        left, right = 0, 0
        res = [0, 0]
        while right < len(nums):
            num = nums[right]
            window += num
            right += 1

                left += 1

```

#### 标准实现（⭐推荐）

```python
        left = cur_sum = max_len = 0
        for right, num in enumerate(nums):
            cur_sum += num
            while cur_sum > target:
                cur_sum -= nums[left]
                left += 1
            if cur_sum == target:
                max_len = max(max_len, right - left + 1)

        return -1 if max_len == 0 else len(nums) - max_len
```

### 5.12 至少有 K 个重复字符的最长子串

- **题目**：[395. 至少有 K 个重复字符的最长子串](https://leetcode.cn/problems/longest-substring-with-at-least-k-repeating-characters)
- **描述**：找到每个字符出现次数 ≥ k 的最长子串长度。
- **示例**：
  - 输入：`s = "aaabb"`, `k = 3`
  - 输出：`3`（"aaa"）

#### 原始实现

```python
# 枚举字符种类数（1~26），配合滑动窗口
class Solution:
    def longestSubstring(self, s: str, k: int) -> int:
        return max([self.longestSubstring_with_n_chrs(s, k, i) for i in range(1, 26 + 1)])

    def longestSubstring_with_n_chrs(self, s, k, n):
        from collections import defaultdict
        window = defaultdict(int)
        left, right = 0, 0
        res = 0
        while right < len(s):
            chr = s[right]
            window[chr] += 1
            right += 1
            while left < right and len(window) > n:
                chr = s[left]
                window[chr] -= 1
                if window[chr] == 0:
                    del window[chr]
                left += 1
            if all(num >= k for num in window.values()):
                res = max(res, right - left)
        return res
```

#### ✅ 标准实现

> **评价**："限制字符种类数 + 滑动窗口"是正确解法之一。另一种常用的是**分治法**：

```python
# 分治法
class Solution:
    def longestSubstring(self, s: str, k: int) -> int:
        if len(s) < k:
            return 0
        from collections import Counter
        count = Counter(s)
        for ch, num in count.items():
            if num < k:
                return max(self.longestSubstring(sub, k) for sub in s.split(ch))
        return len(s)
```

### 5.13 Rabin-Karp 字符串匹配

- **描述**：用滚动哈希在文本中匹配模式串，在 O(n+m) 时间内完成。
- **示例**：
  - 输入：`txt = "hello"`, `pat = "ll"`
  - 输出：`2`

#### 原始实现

```python
def rabinKarp(txt: str, pat: str) -> int:
    L = len(pat)
    R = 256
    Q = 1658598167
    RL = 1
    for i in range(1, L):
        RL = (RL * R) % Q

    patHash = 0
    for i in range(len(pat)):
        patHash = (R * patHash + ord(pat[i])) % Q

    windowHash = 0
    left, right = 0, 0
    while right < len(txt):
        windowHash = ((R * windowHash) % Q + ord(txt[right])) % Q
        right += 1
        if right - left == L:
            if windowHash == patHash:
                if pat == txt[left:right]:
                    return left
            windowHash = (windowHash - (ord(txt[left]) * RL) % Q + Q) % Q
            left += 1
    return -1
```

> **评价**：Rabin-Karp 的核心思想是**滚动哈希**——利用前一个窗口的哈希值计算下一个窗口的哈希值，避免重复扫描。虽在实际中 Python 直接用 `find()` 即可，但滚动哈希的思想在解决某些复杂问题（如重复子串）时非常有用。

```python
        right = p

            if abs(arr[left] - x) <= abs(arr[right] - x):

                left -= 1
            else:

                right += 1

            res.append(arr[left])
            left -= 1
        while right < len(arr) and len(res) < k:
            res.append(arr[right])
            right += 1
        res.sort()
        return res
```

- **示例**：

  - 输出：`[[1,2],[1,4],[1,6]]`

#### 原始实现

```python
class Solution:
    def kSmallestPairs(self, nums1: List[int], nums2: List[int], k: int) -> List[List[int]]:
        import heapq
        pq = []
        for num1 in nums1:
            heapq.heappush(pq, (num1 + nums2[0], num1, nums2[0], 0))
        ret = []
        for _ in range(k):
            _, num1, num2, i = heapq.heappop(pq)
            ret.append([num1, num2])
            if i + 1 >= len(nums2):
                continue
            heapq.heappush(pq, (num1 + nums2[i + 1], num1, nums2[i + 1], i + 1))
        return ret
```

#### ✅ 标准实现

> **评价**：**最小堆 + 横向扩展**是这道题的标准解法。

```python
class Solution:
    def kSmallestPairs(self, nums1: List[int], nums2: List[int], k: int) -> List[List[int]]:
        if not nums1 or not nums2:
            return []
        import heapq
        heap = []
        for i in range(min(k, len(nums1))):
            heapq.heappush(heap, (nums1[i] + nums2[0], nums1[i], nums2[0], 0))
        res = []
        while heap and len(res) < k:
            _, x, y, j = heapq.heappop(heap)
            res.append([x, y])
            if j + 1 < len(nums2):
                heapq.heappush(heap, (x + nums2[j + 1], x, nums2[j + 1], j + 1))
        return res
```
